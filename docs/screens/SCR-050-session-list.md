## States
- **Live:** All data updates via WebSocket (session.started, session.ended, metrics.realtime)
- **New session:** Row appears with slide-in animation + green flash
- **Ended session:** Row fades out

## API References
- API-100: GET /api/v1/sessions
- API-101: GET /api/v1/sessions/stats
- API-102: POST /api/v1/sessions/:id/disconnect
- WebSocket: session.started, session.ended, metrics.realtime
